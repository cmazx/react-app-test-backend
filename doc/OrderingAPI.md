## Pizzario
#### **Ordering API**

Api provides access to public and member area functionality.

1. [**Members area**](#member)
1. [**Order manage**](#order)
1. [**Menu access**](#menu)

#### <a name="menu"></a>Members area
##### Sign up
New clients can be registed by email.
If user already registered we will send him email about he already registered.
Password enter required.
##### Sign in 
Registered clients can sign in to member area using email and password combination.
##### Password restore
Password can be restored.

#### <a name="order"></a>Orders manage
##### History
Allowed only for registered member.
List of client orders sorted by time of create
##### Order repeat
Allowed only for registered member.
Fill client basket by menu positions taken from one of client recent orders.
#### Order cancel
Order can be cancelled if it allowed by order state
#### Order view
Order state page.

#### <a name="order"></a>Menu
##### Categories
List of menu categories with descriptions, and names.
##### Positions
List of positions for requested category.
